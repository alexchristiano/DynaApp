# MERN Stack Project

This project is structured with a `frontend` (React) and `backend` (Express/Node) folder. MongoDB is used for data storage.

## Scripts
- `npm run start`: Starts the backend server
- `npm run client`: Starts the React frontend
- `npm run dev`: Runs both backend and frontend concurrently (requires `concurrently`)

## Environment Variables
- Create a `.env` file in `/backend` with:
  - `MONGO_URI` (MongoDB connection string)
  - `PORT` (server port)

## Structure
- `/frontend`: React app
- `/backend`: Express/Node app
